# MLCC 사이즈 측정 프로그램

반도체 부품인 MLCC는 그 크기가 매우 작기에 정밀함을 요구합니다.<br>
이러한 MLCC에 대해서 적정한 크기의 양품을 선별하고자<br>
영상 처리 기법을 통해 특정 부분의 사이즈를 측정하는 프로그램을 개발했습니다.<br>
<br>
<br>
